In the current situation, it is important to realize that not all people have an equal bargaining position. Specifically, A controls {{p.votes_player_A}} votes,  B controls {{p.votes_player_B}} votes and C controls  {{p.votes_player_C}} votes. As this totals {{p.ABC_coalition_votes}} votes votes, any coalition that controls at least {{p.votes_necessary_for_coalition}} votes controls a majority of the votes. A coalition needs this majority of the votes in order to allocate the  {{p.total_sum_of_outcomes}} points  among the members of the coalition. Thus possible coalitions are coalitions between:

    {% if p.AB_coalition_posibility %}
  • A and B (AB coalition with {{p.AB_coalition_votes}} votes)
    • {% endif %} {% if p.BC_coalition_posibility %}
  • B and C (BC coalition with {{p.BC_coalition_votes}} votes)
    • {% endif %} {% if p.AC_coalition_posibility %}
  • A and C (AC coalition with {{p.AC_coalition_votes}} votes)
    • {% endif %} {% if p.ABC_coalition_posibility %}
  • A, B, and C (ABC coalition with {{p.ABC_coalition_votes}} votes)
    • {% endif %}

Importantly, payoffs can only be allocated to members of a coalition. This means:

    {% if p.AB_coalition_posibility %}
  • AB coalition can allocate the outcomes to A and B (but not to C)
    • {% endif %} {% if p.BC_coalition_posibility %}
  • BC coalition can allocate outcomes to B and C (but not to A)
    • {% endif %} {% if p.AC_coalition_posibility %}
  • AC coalition can allocate outcomes to A and C (but not to B)
    • {% endif %}

If you are included in one of the abovementioned coalitions you will be able to get a share of the outcomes. However, if you are not included in one of the coalitions you will not be able to get a share of the outcomes. Moreover, after the bargaining is over, votes have no value.
